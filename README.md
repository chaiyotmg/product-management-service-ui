# Product Management Service UI

**Vue 3** and **TypeScript**. This project follows the **Atomic Design Methodology**

## Getting Started

### 1. Prerequisites

- Node.js (v22.17.0)

### 2. Setup Environment

Create a `.env` file in the root directory and configure your variables (this example is for local development and testing only):

```env
VITE_API_URL=http://localhost:3000/api
```


### Installation

```sh
npm install
```

### Development

```sh
npm run dev
```

### Production Build

```sh
npm run build
```

### Linting

```sh
npm run lint
```
